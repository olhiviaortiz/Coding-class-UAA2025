Introducción a CSS

¿Qué es CSS?

CSS (Hojas de Estilo en Cascada) es el lenguaje utilizado para dar estilo y presentación a un documento HTML. Permite controlar cómo se muestran los elementos: colores, tipografías, tamaños, márgenes, animaciones, disposición en pantalla, etc.

El término “cascada” significa que las reglas de estilo se aplican siguiendo un orden de prioridad y jerarquía.

Funciones principales de CSS

Estilo visual
Colores, fuentes, tamaños, bordes, sombras.
Maquetación
Posicionamiento de elementos, organización en grillas o flexbox.
Responsividad
Adaptar el diseño a pantallas diferentes (móvil, tablet, escritorio).
Animaciones y transiciones
Movimiento, efectos dinámicos e interactividad.
Accesibilidad y experiencia de usuario
Mejora la legibilidad y la interacción.
